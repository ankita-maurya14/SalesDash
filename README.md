# Sales Dashboard (Power BI)

## Project Overview

This Power BI Sales Dashboard provides a comprehensive analysis of
sales, profit, customer performance, product categories, and payment
methods.

The dashboard is designed to help stakeholders monitor key business
metrics and identify high-performing regions, customers, and products.

------------------------------------------------------------------------

## Key Metrics

-   Total Sales Amount (State-wise analysis)
-   Total Profit (Customer-wise analysis)
-   Monthly Profit Trend
-   Category Distribution
-   Payment Mode Distribution

------------------------------------------------------------------------

## Dashboard Components

### 1. State-wise Sales Analysis

-   Bar chart showing Sum of Amount by State
-   Highlights top-performing states such as Madhya Pradesh,
    Maharashtra, and Uttar Pradesh

### 2. Customer-wise Profit Analysis

-   Displays Sum of Profit by Customer Name
-   Identifies high-value customers

### 3. Monthly Profit Trend

-   Column chart showing Profit by Month (January, February, March)
-   Quarter slicer available (Qtr 1--Qtr 4)

### 4. Payment Mode Distribution

-   Donut chart displaying:
    -   COD
    -   UPI
    -   Debit Card
    -   EMI
    -   Credit Card

### 5. Category Analysis

-   Donut chart for product categories:
    -   Clothing
    -   Electronics
    -   Furniture

### 6. Sub-Category Performance

-   Horizontal bar chart showing category counts:
    -   Stole
    -   Hankerchief
    -   Saree
    -   Bookcases

------------------------------------------------------------------------

## Filters & Interactivity

-   Quarter Selector (Qtr 1--Qtr 4)
-   Dropdown filter
-   Interactive visuals (cross-filtering enabled)

------------------------------------------------------------------------

## Tools Used

-   Power BI Desktop
-   DAX (Data Analysis Expressions)
-   Power Query
-   Excel Dataset

------------------------------------------------------------------------

## Business Insights

-   Certain states contribute significantly higher sales revenue.
-   A small number of customers generate the majority of profits.
-   COD remains a dominant payment method.
-   Clothing category shows higher contribution compared to others.
-   Profit trends vary across months and quarters.

------------------------------------------------------------------------

## How to Use

1.  Open the `.pbix` file in Power BI Desktop.
2.  Use the Quarter filter to analyze specific periods.
3.  Click on any visual to enable cross-filtering.
4.  Explore category and payment trends for deeper insights.
